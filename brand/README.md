# MacroPulse brand kit

The MacroPulse mark is a phase-offset striped sphere. Two discs of equal radius are offset
horizontally and striped in opposite phase; the vertical seam where they meet represents a
**macro-regime transition** — the core idea of the product.

| File | Use |
|------|-----|
| `logo.svg` | Universal — inherits `currentColor` |
| `logo-green.svg` | Primary brand mark (`#3fb85a`) — works on light or dark |
| `logo-mono.svg` | White mark for dark surfaces |
| `logo-black.svg` | Black mark for light surfaces |
| `favicon.svg` | Tight-cropped mark for browser tabs |

## Colors

| Token | Hex | Use |
|-------|-----|-----|
| Green | `#3fb85a` | Primary — MacroPulse |
| Amber | `#f5a623` | IRL Engine accent |
| Indigo | `#4f46e5` | Interactive / links |
| Ink | `#0a0a0a` | Dark surface background |

## Rules

- Keep clear space around the mark equal to one stripe height.
- Don't recolor the stripes outside the palette, rotate the mark, or add effects.
- On photography or busy backgrounds, use the mono variant.
